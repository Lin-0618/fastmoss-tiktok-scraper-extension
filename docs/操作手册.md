# FastMoss TikTok 数据导出扩展操作手册

## 一、工具用途

本工具是一个 Chrome 扩展，用于在 FastMoss 视频来源页批量抓取视频列表数据，并导出 CSV/TXT 文件。

支持页面：

```text
https://www.fastmoss.com/zh/media-source/video
```

主要功能：

- 自动抓取当前列表页数据
- 自动点击下一页并继续抓取
- 导出完整 CSV
- 导出 TikTok 视频链接 TXT
- 导出 TikTok 达人主页链接 TXT
- 自动尝试从 FastMoss 达人详情页、视频详情页补全 TikTok handle 和视频链接

## 二、安装扩展

1. 打开 Chrome 浏览器。
2. 在地址栏输入：

```text
chrome://extensions/
```

3. 打开右上角「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择本工具所在文件夹：

```text
D:\tiktok\工具开发\chrome扩展-爬取数据
```

6. 浏览器右上角会出现「FastMoss TikTok 数据导出」扩展。

## 三、登录 FastMoss

1. 打开 FastMoss 视频来源页：

```text
https://www.fastmoss.com/zh/media-source/video
```

2. 按你的账号正常登录。
3. 等页面表格加载完成。

本扩展不会保存账号密码，也不会代替你登录。它只读取你当前浏览器里已经打开并登录的页面内容。

## 四、开始抓取

1. 打开 FastMoss 视频来源页并确认表格已经加载。
2. 点击 Chrome 右上角的扩展图标。
3. 设置「最多页数」。

建议：

- 测试时先填 `1` 或 `2`
- 正式抓取时再填更大的页数，例如 `50`

4. 点击「开始抓取」。
5. 扩展会自动抓取当前页，然后自动点击下一页继续。
6. 抓取过程中可以点击「停止」。

停止后，已经抓到的数据会保留，可以继续导出。

## 五、导出文件

扩展提供三种导出：

### 1. 下载 CSV

点击：

```text
下载 CSV
```

CSV 包含完整字段，适合用 Excel 打开。

CSV 已经加入 UTF-8 BOM，Windows Excel 直接打开时中文应正常显示。

### 2. 下载视频 TXT

点击：

```text
下载视频 TXT
```

导出一行一个 TikTok 视频链接，例如：

```text
https://www.tiktok.com/@taniamonserratlugoh/video/7628461448393461013
```

如果文件为空，说明当前抓到的数据里还没有成功补全 TikTok 视频链接。

### 3. 下载链接 TXT

点击：

```text
下载链接 TXT
```

导出一行一个 TikTok 达人主页链接，例如：

```text
https://www.tiktok.com/@taniamonserratlugoh
```

## 六、CSV 字段说明

| 字段 | 说明 |
| --- | --- |
| `page` | 抓取到该行时所在页码 |
| `rowIndex` | 当前页内行号 |
| `videoTitle` | 视频标题 |
| `duration` | 视频时长 |
| `creatorName` | 达人昵称 |
| `creatorHandle` | TikTok 达人 handle |
| `tiktokProfileUrl` | TikTok 达人主页链接 |
| `tiktokVideoUrl` | TikTok 视频链接 |
| `fastmossInfluencerUrl` | FastMoss 达人详情页链接 |
| `fastmossVideoUrl` | FastMoss 视频详情页链接 |
| `fastmossVideoId` | FastMoss 页面里的视频 ID |
| `country` | 国家/地区 |
| `category` | 类目/标签 |
| `followers` | 达人粉丝数 |
| `publishedAt` | 发布时间 |
| `views` | 播放量 |
| `sourceUrl` | 抓取来源页面 |

## 七、视频链接生成规则

扩展会按以下顺序尝试生成 TikTok 视频链接：

1. 如果页面或详情页里已经有完整 TikTok 视频链接，直接提取。
2. 如果列表页只有 FastMoss 视频详情链接，例如：

```text
https://www.fastmoss.com/zh/media-source/video/7628461448393461013
```

并且能拿到 TikTok handle，例如：

```text
taniamonserratlugoh
```

则自动拼成：

```text
https://www.tiktok.com/@taniamonserratlugoh/video/7628461448393461013
```

3. 如果缺少 handle，扩展会尝试访问 FastMoss 达人详情页、视频详情页补全。
4. 如果详情页和列表页都没有 handle，则 `tiktokVideoUrl` 会为空，但会保留 `fastmossVideoUrl` 和 `fastmossVideoId`。

## 八、常见问题

### 1. 提示 Receiving end does not exist

原因：扩展刚重新加载后，当前 FastMoss 页面还没有注入脚本。

处理方式：

1. 先在 `chrome://extensions/` 重新加载扩展。
2. 回到 FastMoss 页面刷新一次。
3. 再点击扩展开始抓取。

新版扩展也会自动尝试补注入，通常刷新一次即可。

### 2. CSV 中文乱码

请用新版扩展重新下载 CSV。

新版 CSV 已带 UTF-8 BOM，Windows Excel 可以正确识别中文。旧文件已经乱码的话，需要重新抓取或重新导出。

### 3. TikTok 视频链接为空

常见原因：

- FastMoss 当前列表页没有 TikTok handle
- FastMoss 详情页没有直接暴露 TikTok handle
- 当前账号权限看不到详情页里的外部链接

可以先看 CSV 里的：

```text
fastmossVideoUrl
fastmossVideoId
fastmossInfluencerUrl
```

这些字段能帮助继续排查接口里是否有原始 TikTok 链接。

### 4. 抓取速度变慢

如果某些行缺少 handle，扩展会访问详情页尝试补全，所以速度会比只抓列表慢。

建议先用少量页数测试，例如 `1` 或 `2` 页，确认字段正常后再抓更多页。

### 5. 下一页没有自动继续

处理方式：

1. 确认页面底部有下一页按钮。
2. 确认当前不是最后一页。
3. 刷新 FastMoss 页面后重试。
4. 如果仍然不行，截图分页区域用于调整按钮识别规则。

## 九、更新扩展

如果代码有更新，需要重新加载扩展：

1. 打开：

```text
chrome://extensions/
```

2. 找到「FastMoss TikTok 数据导出」。
3. 点击「重新加载」。
4. 回到 FastMoss 页面刷新一次。

在项目目录中可以运行：

```powershell
npm.cmd test
npm.cmd run check
```

- 解析 FastMoss 视频详情链接和视频 ID
- CSV 导出和中文编码
- TXT 链接导出
- JavaScript 语法检查
